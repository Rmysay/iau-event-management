# Evntly — Event Management System

## Quick Start
```bash
npm install
npm run dev
```

## Folder Structure
```
src/
├── assets/          # Images, fonts, icons
├── components/      # Reusable UI components (EventCard, StatusBadge, FormField)
├── layout/          # Sidebar, TopBar
├── pages/           # Route-level page components
├── features/        # Feature modules (auth, events, registrations)
├── hooks/           # Custom hooks (useEvents, useRegistrations)
├── context/         # React Context (AppContext)
├── services/        # API / data layer (eventService, registrationService)
└── utils/           # Constants, helpers
```
